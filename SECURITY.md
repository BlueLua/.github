# Security Policy

## Supported Versions

Only the latest active release in our repositories is supported with security
updates.

## Reporting a Vulnerability

We take the security of our projects seriously. If you find a security
vulnerability, please do **not** open a public issue. Instead, report it
privately.

Please report security issues by:

- Creating a [private security advisory] (if enabled on the repository)
- Or emailing the maintainers directly (please contact the repository owner or
  organization administrators)

Once a report is received:

1. We will acknowledge receipt of your report within 48 hours.
2. We will investigate the issue and coordinate a fix.
3. We will keep you updated on our progress and notify you when a patch is
   released.

[private security advisory]:
  https://github.com/BlueLua/.github/security/advisories/new
