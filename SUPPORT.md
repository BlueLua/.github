# Support Guidelines

Thank you for using our projects! If you need help, please follow these
guidelines to get support:

## 1. Check Existing Resources

Before asking a question, please make sure to check:

- The official [documentation].
- Existing issues and discussions in the repository where you are experiencing
  the issue.
- Search engines for similar questions.

## 2. GitHub Discussions

For usage questions, discussions, or general help, please use **GitHub
Discussions** in the relevant repository. This keeps our issue trackers focused
on bug reports and feature requests.

## 3. Opening a Question / Support Request

If Discussions are not enabled or you still need help:

- Open an issue using the **Docs / Question** issue template (if available).
- Be as specific as possible: describe what you are trying to achieve, what you
  have tried, and include relevant code examples or error logs.

[documentation]: https://bluelua.github.io
